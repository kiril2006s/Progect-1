
## Опис
Ця програма допомагає студенту вести облік особистих витрат та бюджету.

## Зберігання даних
Дані зберігаються у файлі JSON (data.json).
У файлі зберігається:
- бюджет
- список витрат

При запуску програми дані зчитуються з файлу.
Після кожної зміни файл автоматично оновлюється.

## Основні команди

- допомога — показує список команд
- встановити бюджет — встановлює бюджет
- додати витрату — додає нову витрату
- показати витрати — показує всі витрати
- витрати дата — показує витрати за дату
- витрати період — показує витрати за період
- витрати категорія — показує витрати по категорії
- залишок — показує залишок бюджету
- звіт за категоріями — показує витрати по категоріях
- вийти — завершує програму

## Алгоритм роботи програми

1. Запуск програми
2. Завантаження даних з JSON файлу
3. Виведення вітання
4. Очікування команди користувача
5. Виконання відповідної функції
6. Збереження змін у файл
7. Повторення до команди "вийти"
